# System: Landing page - dance

Welcome to the dance KEG -- a private knowledge base for partner dancing.
Content covers dance styles, techniques, moves, venues, community events,
and instruction notes with a focus on the Twin Cities scene. This note is
the starting point for navigating the KEG.
