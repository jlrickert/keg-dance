# Entity: Guide note structure (guide)

Guide notes provide step-by-step instructions for dance-related goals in
the dance KEG -- practice drills, warm-up routines, technique exercises,
and learning progressions. For flat lookup data, use a [reference](../9).
For summarising external content, use an [article](../11).

## Title format

- `# Guide: DESCRIPTION`

## Minimal template

```md
# Guide: DESCRIPTION

One to three sentence summary of what this guide covers and who it is for.

## Prerequisites

- Familiarity with [move or concept](../NNN)

## Steps

### Step 1: STEP_NAME

Details for this step.

### Step 2: STEP_NAME

Details for this step.

## Related

- [Related note](../NNN)
```

