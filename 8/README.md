# Entity: Concept note structure (concept)

Concept notes explain reusable dance principles, techniques, and ideas in
the dance KEG. Use a concept for broad principles like lead-follow
connection, frame, musicality, or body isolation. For specific named
patterns, use a [move](../16) instead. For lookup tables, use a
[reference](../9).

## Title format

- `# Concept: DESCRIPTION`

## Minimal template

```md
# Concept: DESCRIPTION

One to three sentence summary of the concept and its relevance to
partner dancing.

## Overview

Detailed explanation of the concept.

## Application

How this concept applies across dance styles.

## Related

- [Related note](../NNN)
```

## Metadata template

```yaml
entity: concept
tags:
    - dance
```
