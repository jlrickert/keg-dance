# Entity: System note structure (system)

System notes document how the dance KEG operates -- its conventions,
processes, style rules, and configuration decisions. They are the internal
governance layer that keeps all other notes consistent. See the
[operating reference](../2) for scope and the [entity selection guide](../5)
for when to use this entity type.

## Title format

- `# System: DESCRIPTION - dance`

## Minimal template

```md
# System: DESCRIPTION - dance

One to three sentence summary of the convention or process.

## (sections as needed)

```
