# Rueda de Casino (rueda)

Rueda de Casino is a group salsa format where multiple couples dance in a circle and a caller (the "líder") shouts or
signals moves, causing everyone to execute the same pattern simultaneously, including frequent partner exchanges.

## Key characteristics

- Built on [Cuban salsa](../46) vocabulary and circular movement
- Caller directs the group with named calls (e.g., "Dame", "Enchufla", "Vacílala")
- Partner exchanges ("Dame") are a defining feature
- Typically 4+ couples; the larger the circle the more dynamic it becomes
